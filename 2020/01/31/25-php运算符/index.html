PHP运算符

错误控制符@

当将其放置在一个PHP表达式之前，该表达式可能产生的任何错误信息都将被忽略。

运算符的优先级
递增/递减运算符
  • 递增/递减运算符不影响bool值
  • 递增NULL为1，递减NULL没有效果。

  • 递增/递减在前 先运算后返回，在后 先返回后运算

    逻辑运算符

  • 短路
    true || $b==5
    false && $a ==3
    以上两种都会触发短路效应（后面的不会执行）。

  • (|| &&) > (and or) 优先级不同

    练习：

分析：

  • $c = false || true;
    由于 ||的优先级大于 =(赋值运算符)
    先执行 false || true -> true 然后在赋值给 $c
    故：$c 的值为 true

  • $d = false or true;

    由于 =的优先级大于 or 所以先执行 $d = false;
    故： $d的值为false

    练习：

    解析：

    • （>）> ( || ) > (=) 运算符的优先级
    • 先执行 3>0 –> true
    • $a = true || $b=3>0
    • 发生短路 $b = 3 > 0 不执行
    • $a = true
    • $b = 0

‘==’ ‘===’

‘==’ 值相等就可以

  • 0 0.0 ‘’ ‘0’ array() NULL false 利用 ‘==’ 号都是相等的

‘===’ 类型也要相等
